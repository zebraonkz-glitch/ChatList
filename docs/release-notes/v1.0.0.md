# ChatList v1.0.0

Первый публичный релиз: сравнение ответов нескольких нейросетей из одного промта.

## Что нового

- GUI на PyQt6: промт, параллельная отправка, карточки результатов
- Управление моделями (OpenAI, DeepSeek, Groq, OpenRouter)
- AI-ассистент для улучшения промтов
- История, экспорт Markdown/JSON, светлая и тёмная тема
- Инсталлятор Windows (Inno Setup) с удалением через «Программы и компоненты»

## Установка (Windows)

1. Скачайте `ChatList-1.0.0-setup.exe` из раздела **Assets**.
2. Проверьте SHA256 по файлу `SHA256SUMS.txt`.
3. Установите программу и скопируйте `.env.example` в `.env`.
4. Укажите API-ключи и включите модели в **Приложение → Модели**.

## Файлы в релизе

| Файл | Назначение |
|------|------------|
| `ChatList-1.0.0-setup.exe` | Инсталлятор (рекомендуется) |
| `ChatList-1.0.0.exe` | Портативная сборка |
| `SHA256SUMS.txt` | Контрольные суммы |

## Требования

- Windows 10/11 (x64)
- API-ключи провайдеров (OpenAI, DeepSeek, Groq, OpenRouter и др.)
