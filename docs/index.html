Возможности

  • Параллельная отправка промта в активные модели
  • Сравнение ответов в одном окне
  • AI-ассистент для улучшения промтов
  • История, экспорт MD/JSON, светлая и тёмная тема
  • OpenAI, DeepSeek, Groq, OpenRouter

Быстрый старт

  1. Скачайте и установите ChatList-*-setup.exe
  2. Скопируйте .env.example в .env и укажите API-ключи
  3. Включите модели в Приложение → Модели
  4. Введите промт и нажмите Отправить

Для разработчиков

  • Python 3.11+, PyQt6, SQLite
  • pip install -r requirements.txt
  • python main.py
  • Сборка: .\build.ps1

Релизы публикуются на GitHub Releases. Проверяйте SHA256-контрольные суммы в файле SHA256SUMS.txt.
