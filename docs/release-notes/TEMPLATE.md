## Установка (Windows)

1. Скачайте `ChatList-X.Y.Z-setup.exe` из раздела **Assets**.
2. Проверьте SHA256 по файлу `SHA256SUMS.txt`.
3. Установите программу и скопируйте `.env.example` в `.env`.
4. Укажите API-ключи и включите модели в **Приложение → Модели**.
